# 星号*用法

##函数可变参数*

##函数关键字参数**

##取可迭代对象的可变长元素
确定出可迭代对象中的部分元素，剩下的元素可以用`*+变量名`存起来。

```python
>>> record = ('Dave', 'dave@example.com', '773-555-1212', '847-555-1212')
# 用phone_numbers存放后面剩下的元素
>>> name, email, *phone_numbers = record
>>> name
'Dave'
>>> email
'dave@example.com'
>>> phone_numbers
['773-555-1212', '847-555-1212']

>>> line = 'nobody:*:-2:-2:Unprivileged User:/var/empty:/usr/bin/false'
# fields存中间余下的元素
>>> uname, *fields, homedir, sh = line.split(':')   #分割成一个list
>>> uname
'nobody'
>>> homedir
'/var/empty'
>>> sh
'/usr/bin/false'
>>> fields
['*', '-2', '-2', 'Unprivileged User']

# 取出想要的元素，不要想的通过*号+废弃名称（比如_或者ign）
>>> record = ('ACME', 50, 123.45, (12, 18, 2012))
>>> name, *_, (*_, year) = record
>>> name
'ACME'
>>> year
2012
```

###利用该特性实现递归计算

```python
>>> def sum(items):
...     head, *tail = items
...     return head + sum(tail) if tail else head
...
>>> sum([1,2,3])
6
```
